Aplicación de Clima

Descripción del proyecto

Esta es una aplicación web simple desarrollada en HTML5, CSS3 y JavaScript. Su objetivo es mostrar el clima actual de cualquier ciudad del mundo a través del consumo de una API. Introduciendo el nombre de la ciudad, el usuario puede obtener datos en tiempo real, como la temperatura, la humedad y las condiciones meteorológicas actuales.

Características

  • Consulta el clima en cualquier ciudad del mundo.
  • Datos meteorológicos en tiempo real
  • Interfaz de usuario amigable y receptiva.

Tecnologías utilizadas

  • HTML5: Estructura de la página web.
  • CSS3: Estilos y diseño visual.
  • JavaScript: Lógica para el consumo de la API y manejo de eventos.
  • Postman: para testear los datos devueltos por la api.
  • API: Utilizamos la API de OpenWeather para obtener los datos meteorológicos.
